Neutral roles are used for surfaces and backgrounds, as well as high emphasis text and icons.

Surface variant and on-surface variant can be used for differentiation against a surface.

Outline is a utility color that creates boundaries, visual hierarchy and emphasis to improve usability. The outline color has a 3:1 contrast with all surface colors.
Use the outline color for accessible boundaries around interactive items such as chips, buttons, and text fields.

For components that contain multiple elements, such as cards or dividers, use the Outline variant color.
Outline variant is a utility color that creates boundaries for decorative elements when a 3:1 contrast isn’t required, such as for dividers or decorative elements.
